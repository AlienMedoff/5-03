# Aether-TMA (TON Transaction DSL) 🚀

**Ваш оркестратор ИИ-агентов для Telegram.**

Aether-TMA — это масштабируемый слой для запуска и координации ИИ-агентов. Проект превращает стандартный Telegram-бот в мощный инструмент для работы с данными, предиктивной аналитики и автоматизации.

[Image of client server architecture flow]

## 💡 Что это такое?
В отличие от обычных чат-ботов, Aether — это **оркестратор**. Он связывает Telegram-интерфейс с локальным или облачным ИИ-бэкендом, позволяя агентам выполнять задачи, анализировать рыночные данные и отвечать пользователю в режиме реального времени.

### Основные фичи:
* **Асинхронность:** Полная поддержка `asyncio` и `aiogram 3.x`.
* **Локальный ИИ-бэкенд:** Интеграция с LLM через локальный сервер (порт 8000).
* **Архитектура агентов:** Готовая база для внедрения нескольких специализированных агентов (трейдинг, аналитика, мониторинг).
* **Безопасность:** Использование `.env` для защиты API-ключей.

## 🛠 Архитектура
Система построена на принципе разделения ответственности:
1. **Telegram Interface:** Принимает ввод от пользователя.
2. **Orchestrator (Bot):** Обрабатывает логику и направляет запросы.
3. **Inference Backend:** Мозг системы, где живут модели ИИ.

## 🚀 Быстрый старт

### Требования
* Python 3.10+
* Установленные зависимости

### Установка
1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/ТВОЙ_НИК/aether-multi-agent-bot.git
   cd aether-multi-agent-bot
   ```

2. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Настрой окружение:
   ```bash
   cp .env.example .env
   # Отредактируй .env с твоими API-ключами
   ```

## 🤖 AI модели

### 🟢 Mistral (по умолчанию)
- **Модель:** `mistral-large-latest`
- **Скорость:** ~2.0с
- **Токены:** ~60-80
- **Особенности:** Лучший баланс скорости и качества

### 🟢 Groq
- **Модель:** `llama-3.3-70b-versatile`
- **Скорость:** ~0.6с (самая быстрая)
- **Токены:** ~70-80
- **Особенности:** Ультра-быстрые ответы

### 🟢 Gemini
- **Модель:** `gemini-2.5-flash`
- **Скорость:** ~2.9с
- **Токены:** ~15-20
- **Особенности:** Последняя модель Google AI

## 🚨 Устранение проблем

### Конфликты портов
Если порт 8000 занят, сервис автоматически находит следующий доступный порт (8001, 8002, и т.д.)

### Ошибки моделей
- **Mistral:** Проверь валидность API ключа
- **Groq:** Убедись, что модель `llama-3.3-70b-versatile` доступна
- **Gemini:** Проверь доступ к модели `gemini-2.5-flash`

### Проблемы с Telegram ботом
- Убедись, что токен валиден и не используется где-то ещё
- Проверь подключение к интернету
- Убедись, что AI сервис запущен
